# ✚ Praja Vaidyasala — Smart Clinic System
### ప్రజా వైద్యశాల, అడ్డంకి, ఆంధ్రప్రదేశ్

A complete Smart Clinic Management System built for Praja Vaidyasala, Addanki, Andhra Pradesh.

## Features
- 🏥 Patient Registration & Token Queue
- 👨‍⚕️ Doctor Room — Prescriptions, Blood Tests, X-Ray, Injections
- 💊 Medical Room / Pharmacy
- 🤖 AI Health Bot (Telugu + English)
- 📺 Hall Display
- 📱 WhatsApp / SMS prescription sending
- ☁️ Supabase real-time database sync across all devices
- 📱 Mobile + Desktop responsive

## Live Demo
Hosted on GitHub Pages: `https://YOUR_USERNAME.github.io/praja-clinic/`

## Setup
Just open `index.html` in any browser and click **⚡ Connect Supabase** to link your database.

## Tech Stack
- Pure HTML + CSS + JavaScript (no framework)
- Supabase (PostgreSQL) for real-time data
- Claude AI API for health bot
- Web Speech API for Telugu TTS/STT
